# Taller POO en JavaScript

Este repositorio contiene una serie de ejercicios desarrollados en **JavaScript** con el objetivo de practicar los conceptos básicos de la **Programación Orientada a Objetos (POO)** mediante el uso de funciones constructoras y objetos.

## 📚 Contenido

- **Ejercicio 1:** Gestión de computadores.
- **Ejercicio 2:** Registro de mascotas.
- **Ejercicio 3:** Información y validación de estudiantes.
- **Ejercicio 4:** Administración de libros de una biblioteca.
- **Ejercicio 5:** Simulación del funcionamiento de un vehículo.

## 🛠️ Tecnologías utilizadas

- JavaScript
- Node.js
- Visual Studio Code

## 📖 Conceptos aplicados

- Programación Orientada a Objetos (POO)
- Funciones constructoras
- Creación de objetos
- Métodos
- Propiedades
- Uso de `this`
- Condicionales
- Manipulación de atributos de los objetos
